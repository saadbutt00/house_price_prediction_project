Overview

This Streamlit app predicts house prices based on user inputs. The model is trained using historical house price data and utilizes a Linear Regression algorithm. Users can input features such as square footage, lot size, waterfront presence, view quality, and city to receive a predicted house price.

Features

- Interactive Sidebar: Allows users to input various house features and select a city.
- Predicted Price Display: Shows the predicted house price based on user inputs.
- Dark Theme: Provides a modern and user-friendly interface with a dark theme.

Technologies Used

- Streamlit: For creating the interactive web application.
- Pandas: For data manipulation and processing.
- NumPy: For numerical operations.
- Scikit-Learn: For machine learning and data preprocessing.
- Pickle: For saving and loading the trained model and encoders.
